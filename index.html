EvoFactory

A real-time cell biology simulation game. Start as a single cell, absorb nutrients, build organelles like mitochondria and ribosomes, and evolve through a tech tree of biological upgrades. Core simulation logic written in Rust and exposed to Godot 4 via GDExtension for high-performance gameplay with rich 2D visuals.

Mi-Pueblo

A family location sharing app with real-time geofencing and group management. Family members create circles, set safe zones, and receive alerts when someone arrives or departs. Multi-language support with an Elixir/Phoenix backend handling real-time PubSub, a Flutter mobile client for iOS and Android, and PostGIS for spatial queries and geofence computation.

VoltVoice.io

A Bitcoin donation platform for live streamers. Viewers send tips via the Lightning Network with instant settlement and zero custody risk. Powered by BTCPay Server for payment processing, a Go/Gin backend for WebSocket streaming alerts, and a React/Vite frontend with real-time donation overlays. Fully self-custody — streamers control their own keys.

Honk

A driver appreciation platform. Think "Strava for truck drivers."

What is Honk?

Drivers share trips, mileage, and stats with social features — buddies, leaderboards, and shared location. Verified driving history serves as proof-of-work for reputation.

The platform also includes an ELD aggregator for real-time driver locations and Hours of Service (HOS) data.
